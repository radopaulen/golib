Detailed Description

template<typename DO>
class mc::CVODES< DO >

mc::CVODES is a C++ class for solution of IVPs in ODEs using CVODES and FADBAD++. Besides computing a numerical solution of the IVP, it can also perform first- and directional second-order sensitivity analysis of state functionals in the Mayer form.
