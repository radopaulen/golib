Detailed Description

template<typename T>
class mc::FPRGurobi< T >

mc::FPRGurobi<T> is a C++ class derived from FPRelax<T> that allows definition of factorable programs, generation of linear relaxations for such programs, and solution of these relaxations using Gurobi
