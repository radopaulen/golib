Detailed Description

template<typename T, typename NODE, typename LT_NODE>
class mc::SetInv< T, NODE, LT_NODE >

mc::SetInv is a pure virutal base C++ class implementing a rigorous set inversion algorithm for nonlinear, implicit functions.

REFERENCES:

  • Jaulin, L., and E. Walter, Set inversion via interval analysis for nonlinear bounded-error estimation, Automatica, 29(4):1053--1064, 1993.
  • Lin, Y., and M. A. Stadtherr, Guaranteed state and parameter estimation for nonlinear continuous-time systems with bounded-error measurements, Industrial & Engineering Chemistry Research, 46:7198--7207, 2007.
  • Kieffer, M., and E. Walter, Guaranteed estimation of the parameters of nonlinear continuous-time models: Contributions of interval analysis, Intermation Journal of Adaptive Control & Signal Processing, 25:191--207, 2011.
